🚀 **FinTrack FinanceMS — Admin Panel Codebase**

A modern, scalable, and production-ready **Admin Panel** for the FinTrack Finance Management System, built using **Next.js, Tailwind CSS, and shadcn/ui**. This admin dashboard provides complete control over users, financial data, and system analytics with a premium UI/UX experience.

✨ **Key Features:**

- 🔐 Secure Admin Authentication & Role-Based Access Control
- 📊 Advanced Analytics Dashboard (Income, Expenses, Budgets)
- 👥 User Management (View, Edit, Delete Users)
- 💰 Financial Data Monitoring (Income, Expenses, Transactions)
- 📈 Interactive Charts & Reports (Recharts Integration)
- 🧾 Budget Tracking & Insights
- 🌙 Light & Dark Mode (Theme Switching)
- 📱 Fully Responsive & Modern UI
- ⚡ Optimized Performance with Next.js App Router

🎨 **UI/UX Highlights:**

- Premium collapsible sidebar (shadcn/ui)
- Clean, minimal, and professional dashboard design
- Smooth animations and intuitive navigation

🛠️ **Tech Stack:**

- Next.js (App Router)
- Tailwind CSS
- shadcn/ui + Radix UI
- TypeScript
- Recharts (Data Visualization)
- REST API Integration (Node.js Backend)

🔗 **Use Case:**
Designed for administrators to efficiently manage and monitor the FinTrack ecosystem, ensuring full visibility and control over financial operations.

💡 Built with scalability, performance, and modern design principles in mind.
